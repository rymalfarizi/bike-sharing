## Setup Environment - Anaconda
```
conda create --name main-ds python=3.9
conda activate main-ds
pip install -r requirements.txt
```

## Setup Environment - Shell/Terminal
```
mkdir bike_sharing
cd bike_sharing
pipenv install
pipenv shell
python -m venv myenv
myenv\Scripts\activate
pip install -r requirements.txt
```

## Run steamlit app
```
streamlit run dashboard.py
```
